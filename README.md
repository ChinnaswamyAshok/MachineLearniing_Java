2 - Data Preparation.
3 - Choose a Model.
4 - Train the Model.
5 - Evaluate the Model.
6 - Parameter Tuning.
7 - Make Predictions.

Algorithms:
* DecisionTable
* RandomForest 
* RandomTree
* NaiveBayes
* IBk  
* I48
